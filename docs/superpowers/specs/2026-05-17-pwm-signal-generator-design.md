# PWM 信号发生器 — 设计规格书

**日期:** 2026-05-17
**状态:** 已确认

---

## 1. 概述

基于 STM32F103C8T6 的双通道 PWM 信号发生器，支持 1Hz~100kHz 频率和 0~100% 占空比可调，可接收 FG 信号并显示 RPM。配备 0.96 寸 SSD1306 OLED 屏幕和旋转编码器进行本地控制，同时提供 PC 端 Qt 模拟器用于开发调试和上位机控制。

---

## 2. 系统架构

```
PC 端 (Qt 6 C++ Widgets)
├── 模拟器模式: 独立运行，键盘映射按键
└── 硬件模式: USB CDC 双向通信

STM32F103C8T6 固件 (裸机 C)
├── 菜单引擎 (状态机)
├── PWM 引擎 (TIM1 CH1, TIM4 CH1)
├── FG 采集 (TIM2 输入捕获 + 硬件分频)
└── OLED 驱动 (SSD1306 I2C)

硬件 I/O:
├── PA8  → PWM_CH1 → 74HCT04 → 5V OUT1
├── PB6  → PWM_CH2 → 74HCT04 → 5V OUT2
├── PA0  → FG_IN   (电阻分压 10k+4.7k)
├── PB10 → ENC_A   (旋转编码器)
├── PB11 → ENC_B
├── PB12 → BTN_OK  (按键确认)
└── PB13 → BTN_BACK (按键返回)
```

## 3. OLED 菜单系统

### 3.1 菜单树

```
主界面 (实时概览)
├── CH1设置
│   ├── 频率
│   ├── 占空比
│   └── 输出使能
├── CH2设置
│   └── (同CH1)
├── FG设置
│   ├── 分频比 (/1, /2, /4, /8)
│   └── 显示刷新率
└── 系统设置
    ├── OLED 亮度
    └── 恢复出厂
```

### 3.2 编码器操作

| 操作 | 菜单模式 | 参数编辑模式 |
|------|---------|-------------|
| 旋转 | 上下移动光标 | 增减参数值 |
| 按下 | 进入子菜单/进入编辑 | 确认并退出编辑 |
| 长按(1s) | 返回上一级 | 取消并返回 |
| BACK键 | 返回上一级 | 取消并返回 |

### 3.3 参数编辑加速

- 慢转: ±1 步进
- 快转(>3格/100ms): ±10 步进
- 频率采用对数档位: 1→10→100→1k→10k→100k Hz，每档可微调

## 4. PWM 输出引擎

### 4.1 频率分段自适应 (72MHz 主频)

| 频率段 | 预分频器 | ARR 范围 | 占空比步数 |
|--------|---------|---------|-----------|
| 1~100Hz | 7200 | 100~10000 | ≥100 |
| 100Hz~1kHz | 720 | 100~1000 | ≥100 |
| 1kHz~10kHz | 72 | 100~1000 | ≥100 |
| 10kHz~100kHz | 7 | 102~1028 | ≥100 |

### 4.2 输出安全

- 上电默认关闭所有通道输出
- 独立使能开关

## 5. FG 信号采集

### 5.1 硬件链路

FG_IN (5V) → 电阻分压 10k+4.7k → 3.2V → PA0 (TIM2 CH1)

### 5.2 RPM 计算

```
RPM = (FG_频率 / 分频比) × 60 / 每转脉冲数
```

- 分频比: /1, /2, /4, /8 (硬件分频)
- 默认 2 脉冲/转 (标准 PC 风扇)
- 显示平滑: 500ms 移动平均

## 6. PC 端 Qt 模拟器

### 6.1 模块

- MainWindow: 主窗口，OLED渲染区 + 控制面板
- OLEDWidget: 128×64 像素级仿真，2x 缩放渲染
- MenuEngine: 菜单逻辑，与固件共享代码
- SerialComm: QSerialPort 封装

### 6.2 键盘映射

| 键盘 | 功能 |
|------|------|
| ↑/↓ | 编码器旋转 |
| Enter | 确认 |
| Backspace | 长按返回 |
| Esc | BACK 键 |
| Space | 切换输出使能 |

### 6.3 通信协议

```
帧格式: | 0xAA/0xBB | CMD(1B) | LEN(1B) | DATA(N) | CRC8 |

命令:
- 0x10: 读取状态
- 0x20: 写入PWM参数
- 0x30: 写入FG分频比
- 0x40: 传输OLED缓冲区 (1024字节)
- 0x41: 按键事件
```

### 6.4 两模式

- **模拟器模式**: 独立运行，不依赖硬件
- **硬件模式**: USB CDC 实时通信

## 8. 测试策略

### 8.1 固件

| 测试项 | 方法 | 标准 |
|--------|------|------|
| PWM 频率/占空比 | 逻辑分析仪 | 误差 <1% |
| 5V 电平 | 万用表 | 4.7~5.3V |
| FG 捕获 | 信号发生器注入 | RPM 误差 <0.5% |
| 分频 | FG=60Hz /2→30Hz | 实测=理论 |
| OLED 刷新 | 快速操作 | 无闪烁, <50ms |
| 编码器去抖 | 连转100格 | 无误触发 |
| 上电安全 | 上电测输出 | 默认关闭 |

### 8.2 模拟器

| 测试项 | 方法 | 标准 |
|--------|------|------|
| 独立模式 | 不接硬件操作 | 功能一致 |
| OLED 渲染 | 截图对比 | 像素一致 |
| 串口通信 | 连硬件操作 | 双向同步 |
| 断连恢复 | 拔插 USB | 3s 恢复 |

### 8.3 集成

- PC→USB CDC→STM32→PWM→示波器验证
- PC→PWM→电机→FG→PC 闭环
