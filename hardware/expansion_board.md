# PWM 信号发生器 — 扩展板设计

## 概述

扩展板插入主控板底部 2×7 排针，实现以下功能：
1. **电源通断**: 继电器控制 15V DC 通断，用于被测设备启停
2. **VSP 信号**: DAC 放大电路 (0~3.3V → 0~5V)
3. **FG 转接**: 转速反馈信号直通
4. **6Pin 3.96mm 连接器**: 对外连接被测设备

## 电源链路

```
220V AC ──→ KBP210 整流桥 ──→ 310V DC ──→ 400V/47μF 滤波
                                            │
                                    高压 DC-DC 模块
                                    (输入 85~265V AC, 输出 24V DC)
                                            │
                                       24V DC
                                            │
                                    LM2596-ADJ 降压
                                    (输入 24V, 输出 15V, 可调)
                                            │
                                       15V DC
                                            │
                              ┌─────────────┤
                              │             │
                          [继电器触点]    直通
                          (常开, PB5控制)   │
                              │             │
                              ▼             │
                         Pin4 (15V OUT)     │
                         (给被测设备)       (运放VCC)
```

> **注意**: 310V DC 不能直接用 XL2596 等低压 DC-DC 降压。需用高压输入电源模块（如 HLK-2M24, AC-DC 模块, 输入 85~265V AC, 输出 24V DC），再用 LM2596 降到 15V。

### 替代方案

| 方案 | 元件 | 输入 | 输出 | 功率 | 成本 |
|------|------|------|------|------|------|
| 方案A (推荐) | HLK-2M24 AC-DC 模块 + LM2596 | 85~265V AC | 24V → 15V | 2W | ¥8 |
| 方案B | 高压 DC-DC 模块 (5~400V 输入) | 310V DC | 15V | 1~3W | ¥15 |
| 方案C | 工频变压器 + 整流 + 稳压 | 220V AC | 15V | 任意 | ¥20+ |

推荐方案A: HLK-2M24 体积小、安全隔离、价格低。

## 继电器驱动电路

```
              3.3V
               │
              [1kΩ]
               │
PB5 ────[1kΩ]──┤──→ S8050 基极 (NPN)
               │        │
              GND      集电极 ──→ 继电器线圈负端
                              │
                             发射极 ──→ GND

继电器线圈正端 ──→ 5V (或 3.3V, 取决于继电器规格)

继电器触点 (常开):
  15V 输入 ──→ 触点公共端
  触点常开端 ──→ Pin4 (15V OUT)
  (PB5 高电平时吸合, 15V 导通)

续流二极管 1N4007:
  阳极 ──→ 继电器线圈负端 (集电极端)
  阴极 ──→ 继电器线圈正端 (5V 端)
```

### 继电器选型

| 参数 | 要求 | 推荐型号 |
|------|------|---------|
| 线圈电压 | 5V (或 3.3V) | SRD-05VDC-SL-C |
| 触点容量 | ≥10A / 250VAC | SRD-05VDC-SL-C (10A) |
| 封装 | 1 脚 PCB 直插 | SRD-05VDC |

> 如果用 5V 继电器，线圈从主控板 5V 引脚供电。PB5 驱动 NPN 三极管 S8050 控制通断。

## DAC 放大电路 (3.3V → 5V)

```
PA4 (DAC 0~3.3V)
  │
  ├──→ R1 (10kΩ) ──→ GND
  │
  └──→ MCP6002 同相输入 (+)
            │
           (+) OP ──→ VSP_OUT (0~5V)
            │
           (-) ──┬──→ R2 (5.1kΩ) ──→ VSP_OUT
                  │
                 R3 (10kΩ)
                  │
                 GND

增益 = 1 + R2/R3 = 1 + 5.1/10 = 1.515
3.3V × 1.515 ≈ 5.0V

MCP6002:
  VCC ──→ 5V (从主控板引出)
  GND ──→ GND
  输出 ──→ Pin5 (VSP)
```

### 运放选型

| 型号 | 类型 | VCC | 带宽 | 封装 | 成本 |
|------|------|-----|------|------|------|
| MCP6002 | 轨到轨, CMOS | 1.8~6V | 1MHz | SOT-23-8 | ¥1.5 |
| LMV358 | 低压, CMOS | 2.7~5.5V | 1MHz | SOP-8 | ¥0.8 |
| OPA2340 | 精密, 轨到轨 | 2.7~5.5V | 5.5MHz | SOP-8 | ¥3 |

推荐 MCP6002: 轨到轨输出, 单电源供电, SOT-23-8 封装小。

## 6Pin 连接器 (3.96mm)

扩展板对外接口，连接被测设备。

```
        ┌─────────────────────────┐
        │  6Pin 3.96mm 插拔式座子  │
        ├───┬───┬───┬───┬───┬───┤
        │ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │
        ├───┼───┼───┼───┼───┼───┤
        │310V│NC │GND│15V│VSP│FG │
        └───┴───┴───┴───┴───┴───┘
```

| Pin | 信号 | 说明 |
|-----|------|------|
| 1 | 310V | 整流后高压直流 (给被测设备的变频器/电机驱动等) |
| 2 | NC | 空脚 (预留) |
| 3 | GND | 共地 |
| 4 | 15V | 继电器控制后的 15V DC (给被测设备电源) |
| 5 | VSP | 0~5V 模拟电压 (DAC 放大后, 用于变频器速度控制) |
| 6 | FG | 转速反馈信号 (直通到主控板 BNC3/PA0) |

### Pin6 (FG) 直通连接

```
Pin6 (FG) ──→ 扩展排针 PA0 (已有分压电路在主控板上)
```

主控板上 PA0 已有 10k+10k 分压 (5V→2.5V)，扩展板上 FG 信号直接从 Pin6 连到排针 PA3 或直接跳线到主控板 BNC3。

## 扩展排针连接关系

扩展板插入主控板底部 2×7 排针后，使用的引脚：

| 主控排针 | STM32 引脚 | 扩展板功能 |
|----------|-----------|-----------|
| Pin 1 | PA3 | FG 信号输入 (可选, 或直接用 BNC3) |
| Pin 2 | PA4 | DAC VSP 输出 → 运放放大 |
| Pin 6 | PB5 | 继电器控制 |
| Pin 12 | 5V | 电源 (继电器线圈 + 运放 VCC) |
| Pin 13 | GND | 共地 |
| Pin 14 | GND | 共地 (双地脚) |

其余排针 (PA5, PB0, PB1, PB7, PB9, PB10, PB11, PB12) 本次不使用，保留给未来扩展。

## 安全注意事项

1. **高压危险**: 310V DC 和 220V AC 部分必须做隔离处理，PCB 布线时高压区域与低压区域保持 ≥6mm 间距
2. **整流桥散热**: KBP210 在 2A 负载下发热明显，需加散热片或预留铺铜散热
3. **继电器触点容量**: SRD-05VDC 触点容量 10A/250VAC，被测设备电流不得超过此值
4. **共地**: 扩展板 GND 与主控板 GND 必须可靠连接，否则 DAC 和 FG 信号会异常
5. **先断电再插拔**: 扩展板带电插拔可能损坏主控板或被测设备
