## PWM 频率分段策略

```
目标频率 Freq → 自动选择预分频器:

  ┌────────────────────────────────────────────────────┐
  │  Freq ∈ [1, 100] Hz    → PSC = 7199  (10kHz 计数) │
  │  Freq ∈ [100, 1k] Hz   → PSC = 719   (100kHz 计数)│
  │  Freq ∈ [1k, 10k] Hz   → PSC = 71    (1MHz 计数)  │
  │  Freq ∈ [10k, 100k] Hz → PSC = 7     (9MHz 计数)  │
  └────────────────────────────────────────────────────┘

  ARR = 计数时钟 / Freq - 1
  CCR = ARR × Duty% / 100
```

## 通信协议帧格式

```
┌──────────┬─────┬─────┬──────────┬──────┐
│ Header   │ CMD │ LEN │ DATA     │ CRC8 │
│ 1 byte   │ 1B  │ 1B  │ 0~N byte │ 1B   │
├──────────┼─────┼─────┼──────────┼──────┤
│ 0xAA     │     │     │          │      │  PC → MCU
│ 0xBB     │     │     │          │      │  MCU → PC
└──────────┴─────┴─────┴──────────┴──────┘

CRC8-CCITT: poly=0x07, init=0x00
计算范围: HEADER + CMD + LEN + DATA
```
